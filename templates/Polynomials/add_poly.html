What is the Addition of Polynomials?

In general, a polynomial is defined as an expression having different terms. Adding polynomials means combining the similar terms (terms with the same order). To find the sum of polynomials add the coefficients of variables having the same power. We have two different methods to add polynomials; they are like term method and column method.

How to add polynomials?

Steps to Add Polynomials

Following are the steps to add two polynomials easily.

  • Take any two polynomials having different terms.
  • Place the like terms together.
  • Add all like terms and constants.
  • Write the final polynomial either in ascending or descending order of their degree.

If you are n't aware of the concept of the like terms you can add polynomials using column method i.e. line the polynomials up column-wise and add. Using the columns method can be great for complicated sums.

Multiplying Polynomials with Example

Example

Question:Add 1 + 3x² + 5x + 10x³ + 2x + x⁴ + x³ + 5

Solution:

Given input is 1 + 3x² + 5x + 10x³ + 2x + x⁴ + x³ + 5

Group the similar terms

1 + 3x² + 5x + 10x³ + 2x + x⁴ + x³ + 5 = 1 + 5 + 5x + 2x + 3x² + 10x³ + x³ + x⁴

Add the like terms

= 6 + 7x + 3x² + 11x³ + x⁴

Therefore, 1 + 3x² + 5x + 10x³ + 2x + x⁴ + x³ + 5 = 6 + 7x + 3x² + 11x³ + x⁴.

FAQs on Adding Polynomials

1. How do you add polynomials?

To add polynomials, remove the parentheses or braces between them. Write similar terms together. Add similar terms and constants to get the final expression.

2. What are polynomials?

In mathematics, polynomials are expressions that are composed of several terms. In the word polynomial, the term "poly" means many, "nomial" means terms.

3. Solve x²+2-6x²+7x-9.

x²+2-6x²+7x-9 = x²-6x²+7x-9+2 = -5x²+7x-7 .

4. How to add polynomials using the column method?

Adding polynomials using the column method means writing two different polynomials in two different rows one after the other. The like terms of both polynomials should be in the same column and then add them.
